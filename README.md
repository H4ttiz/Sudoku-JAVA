# Sudoku-JAVA

Implementação do clássico jogo **Sudoku** em Java, com duas versões diferentes:

* 🖥️ Versão executada no **terminal**
* 🎨 Versão com **interface gráfica**

O projeto foi desenvolvido com o objetivo de praticar **lógica de programação**, manipulação de estruturas de dados e conceitos modernos da linguagem Java.

## 🎮 Funcionalidades

* Execução do jogo via **terminal**
* Versão com **interface gráfica**
* Validação das jogadas
* Estruturação do tabuleiro
* Manipulação de entradas do usuário

## 🧠 Conceitos praticados

Durante o desenvolvimento foram aplicados diversos recursos da linguagem:

* Lógica de resolução e validação de Sudoku
* Programação Orientada a Objetos
* **Java Collections**
* **Stream API**
* **Expressões Lambda**
* Estruturação de interface gráfica em Java

## 🛠️ Tecnologias utilizadas

* Java
* Java Swing (interface gráfica)
* Collections Framework
* Stream API
* Lambda Expressions

## 📂 Estrutura do projeto

```
src/
 ├── game
 ├── ui
 ├── utils
 └── main
```

## 🎯 Objetivo do projeto

Explorar a implementação de um jogo clássico para exercitar:

* raciocínio lógico
* organização de código
* manipulação de estruturas de dados
* criação de interfaces gráficas em Java
